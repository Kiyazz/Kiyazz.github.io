KeyEngine

2D Game engine implemented in C++ for general-purpose games. Based of the A2 engine API with additional support for saving, including emulator-like save states

Uses Lua as a scripting language, SDL for windowing and rendering, box2d for physics, alongside luabridge, rapidjson, and glm libraries internally

Open-Source, download the engine and build scripts from the Github!
